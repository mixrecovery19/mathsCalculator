Euler's Totient Function φ(n)

Euler's Totient Function, written as \( \phi(n) \), counts how many positive integers less than or equal to n are relatively prime (coprime) to n.

What is Euler's Totient Function?

Before we can use Euler's Totient Formula, we first need the prime factorisation of the number.

For example:

\[196 = 2 \times 2 \times 7 \times 7\]

Euler's Totient Formula does not use every prime factor. Instead, it only uses each distinct prime once.

\[2,\;7 \]

These distinct prime values are then substituted into Euler's Totient Formula:

\[ \phi(196)=196 \left(1-\frac12\right) \left(1-\frac17\right) \]

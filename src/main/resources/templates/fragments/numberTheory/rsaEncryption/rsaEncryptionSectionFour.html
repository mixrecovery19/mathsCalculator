4. Calculate the Private Exponent (d)

How we calculate the private exponent, the \(d\) value

The private exponent, \(d\), is the second part of the RSA key pair. It is chosen so that: \[ ed \equiv 1 \pmod{\phi(N)} \] For educational examples using small numbers, one convenient way to calculate \(d\) is by repeatedly testing the expression: \[ d=\frac{\phi(N)x+1}{e} \] Begin by letting \(x=1\). Multiply \(\phi(N)\) by \(1\), add \(1\), then divide the result by your chosen value of \(e\). If the answer is not a whole number, increase \(x\) to \(2\) and repeat the calculation. Continue increasing \(x\) by \(1\) until the result produces a whole number. That whole number is your value of \(d\).
This simple trial-and-error approach is ideal for the small numbers used in educational demonstrations. Real RSA implementations use much more efficient algorithms to calculate \(d\) when working with very large prime numbers.

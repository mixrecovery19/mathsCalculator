RSA Encryption Walkthrough

RSA is one of the world's most widely used public-key cryptographic systems. Rather than simply calculating the answer, this walkthrough demonstrates each stage used to generate the public and private keys before showing how a message can be encrypted and decrypted using small numbers suitable for learning.

About this Walkthrough

During this walkthrough you will:

  • Select two prime numbers (p₁ and p₂).
  • Calculate \[N = p_1 \times p_2\]
  • Calculate Euler's Totient \[\phi(N) = (p_1 - 1)(p_2 - 1)\]
  • Select a suitable public exponent \[e\] where \[\gcd(e,\phi(N)) = 1\]
  • Calculate the private exponent \[d\]
  • Create the public key \[Pu(e,N)\]
  • Create the private key \[Pr(d,N)\]
  • Encrypt a message \[C = M^e \pmod N\]
  • Decrypt the ciphertext \[M = C^d \pmod N\]

This walkthrough is designed for educational purposes and intentionally uses small prime numbers so that each stage of the RSA algorithm can be performed manually. Real RSA systems use extremely large prime numbers, but the smaller values used here allow each calculation to be followed step by step.

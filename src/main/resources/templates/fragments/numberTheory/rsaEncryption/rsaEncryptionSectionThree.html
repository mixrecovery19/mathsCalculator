3. Select the Public Exponent (e)

The public exponent, \(e\), forms part of your RSA public key. For RSA to work correctly, the value of \(e\) must be coprime with \(\phi(N)\). In other words: \[ \gcd(e,\phi(N))=1 \] This walkthrough will verify that the value you enter satisfies this requirement. If you would like to see how two numbers are tested for coprimality, refer to the Euclidean Algorithm walkthrough in the Number Theory section.

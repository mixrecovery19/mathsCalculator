Inverse Matrix Formula

The inverse of a matrix is calculated using:

\[ A^{-1}=\frac{1}{\det(A)}\times\operatorname{adj}(A) \]

Where:

  • \(A^{-1}\) = the inverse matrix
  • \(\det(A)\) = the determinant of the original matrix
  • \(\operatorname{adj}(A)\) = the adjugate matrix

In this walkthrough, you first calculated the determinant of the original matrix. You then constructed the adjugate matrix by calculating the minor for each position. Finally, each value in the completed adjugate matrix is divided by the determinant to produce the inverse matrix.
